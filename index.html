Comment récupérer vos données ChatGPT

  • Connectez-vous à votre compte ChatGPT
  • Cliquez sur votre profil
  • Accédez aux paramètres de votre compte
  • Dans "Data controls", cliquez sur "Export data"
  • Attendez de recevoir un email de confirmation
  • Téléchargez vos données depuis le lien fourni
  • Dézipper le fichier qui vous a été envoyé
  • Cherchez le fichier conversations.json dans le dossier
